# Json2Nim

Json2Nim is a program that converts JSON schema into Nim types.

## Installation

Use git to pull the repository locally, then build it using the following:

```bash
git clone https://github.com/isaacdonaldson/json2nim.git
cd json2nim
nimble build
```

## Usage

```bash
json2nim -h
```
